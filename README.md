#Grade Calculator

🔹 Description:
This Python program takes a student's score as input and assigns a grade based on predefined ranges.

🔹 Features:
- Accepts numeric input
- Uses conditional statements to classify grades
- Handles invalid input values
- Demonstrates range-based decision making

🔹 Technologies:
- Python

🔹 Purpose:
Created to practice conditional logic and decision-making in Python.
